# ALANA Codes Website

A modern, responsive website for ALANA Codes, showcasing AI-driven data compression technology.

## Features

- **Modern Design**: Clean, professional design that reflects the technical nature of your business
- **Responsive Layout**: Works perfectly on all devices (desktop, tablet, mobile)
- **Interactive Elements**: Smooth animations, hover effects, and dynamic content
- **Professional Presentation**: Effectively communicates your compression technology and business value

## Website Sections

1. **Hero Section**: Eye-catching introduction with key statistics and compression flow visualization
2. **Problem Section**: Clear explanation of the data processing challenges you solve
3. **Solution Section**: Three key benefits of your AI-powered compression technology
4. **Technology Section**: Detailed breakdown of your three core technologies (SW Core, FPGA Core, Microprocessor Integration)
5. **Use Cases**: Specific applications in satellites, drones, video, and images
6. **Partners**: Strategic partnerships with ASTRA and IAI
7. **Contact**: Professional contact information for Daniel and Russlan

## Technical Features

- **Smooth Scrolling**: Navigation links smoothly scroll to sections
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Animations**: Cards animate in as users scroll down the page
- **Counter Animation**: Hero statistics animate from 0 to target values
- **Parallax Effects**: Subtle background movement for visual interest
- **Active Navigation**: Current section is highlighted in the navigation

## How to Use

1. **Open the website**: Simply open `index.html` in any modern web browser
2. **Navigate**: Use the top navigation menu to jump to different sections
3. **Mobile**: The website automatically adapts to mobile devices with a hamburger menu
4. **Customize**: Edit the HTML, CSS, or JavaScript files to make changes

## Customization Options

### Colors
- Primary Blue: `#2563eb` (used for headings and accents)
- Secondary Purple: `#764ba2` (used in gradients)

- **Text**: Edit content directly in the HTML file
- **Images**: Replace Font Awesome icons with your own images
- **Contact Info**: Update contact details in the contact section
- **Statistics**: Modify the compression ratios and percentages in the hero section

### Styling
- **Fonts**: Currently uses Inter font family (easily changeable in CSS)
- **Layout**: Grid-based responsive design that adapts to content
